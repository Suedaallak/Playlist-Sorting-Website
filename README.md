# Playlist Sorting Tool

This project is a web-based playlist sorting application developed using HTML, CSS, and JavaScript.

## Features
- Load predefined playlists (small and large)
- Create and manage custom playlists
- Add and delete songs dynamically
- Sort songs by title, artist, duration, or listened count
- Choose between Bubble Sort and Merge Sort
- Ascending and descending sorting order
- Display execution time for sorting algorithms

## Technologies Used
- HTML
- CSS
- JavaScript
- Git & GitHub

## Purpose
This project was created as part of an academic assignment to demonstrate:
- Array-based data structures
- Sorting algorithms
- Algorithm performance comparison
- Basic web application design
